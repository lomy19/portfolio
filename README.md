# portfolio

Source du site portfolio de Lory Morello.

Ce projet est une application React (Create React App). Pour lancer le projet en local :

```powershell
npm install
npm start
```

Le site contient les sources du portfolio (dossier `src`) ainsi que les assets dans `public/assets`.

Déployez le dossier `build` produit par `npm run build` sur le service d'hébergement de votre choix.
